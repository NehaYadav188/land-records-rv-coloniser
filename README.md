# 🏘️ RV Coloniser - Land Records Management System

A modern, mobile-responsive web application for managing land records and property listings.

## ✨ Features

### User Portal
- 📱 **Mobile-Responsive Design** - Works seamlessly on all devices
- 🏠 **Property Listings** - Browse available land plots with detailed information
- 🔍 **Advanced Search & Filters** - Find properties by location, status, and more
- 📄 **Pagination** - Easy navigation through multiple properties
- 💬 **WhatsApp Integration** - Direct contact with detailed property information
- 📞 **Multiple Contact Options** - WhatsApp, Call, and Site Visit booking
- 💼 **Financial Advice** - Get expert guidance on loans and payment plans
- 🎯 **Deal Notifications** - Popup alerts for freshly added properties (limited to 3 shows)
- 🏡 **Motivational Banner** - "Make Your Dream Home Come True"

### Admin Portal
- 🔐 **Secure Authentication** - Session-based login with 1-hour timeout
- 📊 **Dashboard** - Comprehensive overview of all land records
- ➕ **Add/Edit/Delete** - Full CRUD operations for lands and plots
- 🔄 **Transaction Types** - Track sold, purchased, or both
- 📏 **Area Management** - Multiple units (sqft, sqm, yard) with auto-conversion
- 💰 **Financial Tracking** - Auto-calculated balances and payment details
- ✅ **Form Validation** - Comprehensive validation for all inputs
- 🔍 **Advanced Filters** - Filter by status, possession, balance, transaction type
- 📱 **User Site Navigation** - Quick access to user portal without session loss
- 🗑️ **Land Deletion** - Remove entire land records with confirmation

## 🚀 Quick Start

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/land-records-app.git

# Navigate to project directory
cd land-records-app

# Install dependencies
npm install

# Start development server
npm start
```

The app will open at [http://localhost:3000](http://localhost:3000)

- **Styling**: Tailwind CSS 3.3.0
- **Routing**: React Router DOM 6.30.3
- **Icons**: Lucide React

## 🎨 Design Features

- **Color Scheme**: Emerald-Teal-Cyan gradient
- **Responsive Breakpoints**: Mobile (< 640px), Tablet (640-1024px), Desktop (> 1024px)
